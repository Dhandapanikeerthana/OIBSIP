# Online Reservation System

## Oasis Infobyte Internship

### Description
A Java Swing-based Online Reservation System developed using Java Swing and SQLite.

### Features
- User Login
- Train Reservation
- Ticket Cancellation
- PNR Generation
- Database Integration

### Technologies Used
- Java
- Swing
- JDBC
- SQLite
